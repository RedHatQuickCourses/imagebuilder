Guided Exercise: Create an ISO image using the composer CLI

Create an ISO image with selinux package using the composer CLI.

Outcomes

  • Create an ISO image with a selinux package using the image builder CLI.

    1. Create a blueprint called image-installer-test by using composer-cli.

    2. Add the selinux-policy-targeted package.

    3. Create a .iso image from the image-installer-test blueprint.

Instructions

  1. Open a terminal on workstation or connect to it via SSH. Create a image-installer-test.toml file with the following content. name = "image-installer-test" description = "image-installer-test" version = "0.0.1" modules = [] groups = []

    [[packages]]
name = "selinux-policy-targeted"

  2. Push the image-installer-test.toml file to create a blueprint.

    [student@workstation ~]$ composer-cli blueprints push image-installer-test.toml

  3. List the blueprints and ensure that the image-installer-test blueprint is listed.

    [student@workstation ~]$ composer-cli blueprints list
composer-test
composer-test-new
image-installer-test

  4. Display the packages needed to install the blueprint.

    [student@workstation ~]$ composer-cli blueprints depsolve image-installer-test
blueprint: image-installer-test v0.0.1
    audit-libs-3.0.7-101.el9_0.2.x86_64
...output omitted...
selinux-policy-targeted-34.1.29-1.el9_0.2.noarch
...output omitted...

  5. Start a compose using the image-installer-test blueprint and output type as image-installer.

    1. For .iso image the output type used is image-installer. The CLI names are mentioned in the table at 1.2. RHEL image builder output formats

      [student@workstation ~]$ composer-cli compose start image-installer-test image-installer
Compose 9dd254f5-724e-4479-bf94-1c1f9e9bced4 added to the queue

    2. The ID in the preceding output is different on your system.

  6. Wait for the image to be created before continuing. This can take about 10-15 mins. To check the status, run the composer-cli compose status command.

    [student@workstation ~]$ composer-cli compose status
ID Status Time Blueprint Version Type Size

    [student@workstation ~]$ composer-cli compose status

  7. Download the iso image from the CLI.

    [student@workstation ~]$ composer-cli compose image 9dd254f5-724e-4479-bf94-1c1f9e9bced4
9dd254f5-724e-4479-bf94-1c1f9e9bced4-installer.iso: 3075.94 MB

    1. The image ID is different on your system. Replace it before you run the command.

  8. List the downloaded iso.

    [student@workstation ~]$  ls -al *.iso
-rw-------. 1 student weldr 3225354240 Oct 27 09:53 9dd254f5-724e-4479-bf94-1c1f9e9bced4-installer.iso

  9. Run the grading script to verify your work.

    [student@workstation ~]$ lab grade imagebuilder-cli
